# 自适应实时数据清洗策略研究（Adaptive Real-Time Data Cleaning Strategy）

## 项目简介

本项目为课程论文“自适应实时数据清洗策略研究：基于数据状态感知的在线方法”的实验实现与数据分析代码库。项目旨在针对实时数据流中的缺失值与异常值问题，提出一种基于滑动窗口和数据状态感知机制的自适应清洗方法，实现在线数据修复与异常抑制。通过多组模拟实验验证了该方法在数据连续性、稳定性以及异常抑制效果方面的优势。

## 功能特点

* **自适应清洗策略**：根据数据波动和异常频率动态调整清洗参数。
* **缺失值与异常值处理**：支持均值、中位数填补及基于阈值或滑动窗口的异常修复。
* **实验可视化**：提供多种实验结果可视化图表，直观展示不同策略的性能对比。
* **参数敏感性分析**：支持滑动窗口大小、异常阈值、缺失比例等参数的实验探索。

## 实验说明

本项目包含主实验与五个子实验，实验数据由 Python 随机生成，模拟实时数据流。各实验设计如下：

1. 主实验：验证自适应清洗策略在整体数据流下的效果。
2. 子实验 1：不同异常比例下的清洗性能分析。
3. 子实验 2：不同缺失率下的清洗鲁棒性分析。
4. 子实验 3：滑动窗口长度敏感性分析。
5. 子实验 4：异常阈值参数影响分析。
6. 子实验 5：自适应策略与固定规则策略的对比分析。

## 运行指南

1. 克隆本仓库：

```bash
git clone https://github.com/yourusername/adaptive-data-cleaning.git
cd adaptive-data-cleaning
```

2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 运行实验：

```bash
python main_experiment.py        # 主实验
python sub_experiment1.py        # 子实验1
python sub_experiment2.py        # 子实验2
# 依次类推
```

4. 实验结果将自动生成图表并保存在 `results/` 文件夹中。

## 环境依赖
